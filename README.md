# Racket-MUD

Racket-MUD is an application for hosting a Multi-User Dimension, or MUD.

It is in early early early alpha.

## Getting Started

To get started with Racket-MUD, clone the repo and from its directory, `racket main.rkt`

### Prerequisites

- Racket
